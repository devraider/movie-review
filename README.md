## How to Get and Run the Application 🛠️

### 1. Clone the application

```bash
git clone git@github.com:devraider/movie-review.git
cd movie-review
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Run application in developemnt mode

```bash
npm start
```

### 4. Build for Production

```bash
npm run build
```
